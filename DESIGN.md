# Veroflower — Design Guidelines

## Color Palette
- **Primary**: Blush pink (#F4C2C2 / rose-200 range)
- **Accent**: Soft sage green (#A8C5A0 / soft green)
- **Background**: Warm white (#FFFAF9)
- **Text**: Deep charcoal (#2D2D2D) for headings, soft gray (#6B6B6B) for body
- **CTA Button**: Blush pink background, white text, soft rounded corners

## Typography
- **Headings**: Serif font (e.g., Playfair Display or similar elegant serif) — large, airy, feminine
- **Body**: Clean sans-serif (e.g., Inter or Lato) — readable, light weight
- **CTA**: Medium weight sans-serif, all-caps or sentence case

## Layout & Spacing
- Full-width sections, generous padding
- Minimal — no clutter, lots of white space
- Mobile-first responsive layout

## Hero Section
- Full-height or near-full-height section
- Large, beautiful floral background or prominent flower imagery (use CSS gradients + decorative elements if no real images)
- Centered or left-aligned headline: e.g., "Fresh Flowers, Delivered with Love"
- Short tagline below: e.g., "Handcrafted bouquets for every moment"
- Bold CTA button: "Visit Us"

## Store Info Section
- Clean, centered layout
- Store name, address, and business hours displayed clearly
- Soft background (light blush or white)
- Minimal icons or elegant dividers

## Elevation & Components
- Subtle box shadows on cards if any
- Soft border-radius (8–16px) on buttons and cards
- No harsh lines — everything should feel soft and organic

## Overall Vibe
Feminine, modern, luxurious but approachable. Like a high-end boutique florist Instagram page translated into a landing page.
